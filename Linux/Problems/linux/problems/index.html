    CUDA driver version is insufficient for CUDA runtime version

    • NVIDIA GPU的驱动程序版本查看命令：nvidia-smi
    • CUDA runtime version（运行时版本）是在python中安装的cudatoolkit和cudnn程序包的版本
    • 查看命令：pip list
    • 查看cuda当前版本
    cat  /usr/local/cuda/version.txt 
nvcc -V

    虚拟机不显示ip地址的解决方法

    • 在root下：dhclient -v;或者ubuntu下：sudo dhclient -v
    • 然后即可查看ip:ifconfig
    • 远程vpn虚拟机ping不通，很有可能是因为虚拟机没有联网。

    “Sorry, Ubuntu 20.04 has experienced an internal error”

    • 解决方法：
    sudo rm -r /var/crash/*

    failed to start ssh.service unit not found

    • sudo service ssh restart打不开Centos7的ssh
    • 是因为service ssh restart是Centos6的命令。Centos7的命令应该是:
    systemctl restart sshd
    • 参考 https://blog.csdn.net/wo541075754/article/details/79092281

    GULP./mkgulp编译错误：

    gfortran: error: unrecognized command line option ‘-ffpe-summary=invalid,zero

    • 主要是gcc编译器太老了，更新一下就好；

    • Cenos7更新方法：

      gcc --version #查看当前版本
sudo yum install centos-release-scl
sudo yum install devtoolset-7-gcc*
scl enable devtoolset-7 bash
which gcc
gcc --version #查看更新后版本

    • 再次编译./mkgulp

    • 参考：https://www.cnblogs.com/Hi-blog/p/How-To-Update-Gcc-On-CentOS7.html
